## Security Hub - Dashboard

#### Security Hub summary

1. Go to the Summary page in Security Hub to get an overview of your security posture. [AWS Security Hub summary page](https://console.aws.amazon.com/securityhub/home?#/summary)

2. At the top, check out the **Security standards** widget. It shows your latest security score and scores for each standard. These scores (0–100%) reflect how many controls have passed out of all enabled controls.
   ![VPC](/images/2/2.1-AWS-Security-Hub/2.1.2-Security-Hub-Dashboard/s2.png)
3. Take a moment to review the insights on the Summary page. Can you spot which resources are failing the most checks? There’s a widget that breaks down findings by resource type.
   ![VPC](/images/2/2.1-AWS-Security-Hub/2.1.2-Security-Hub-Dashboard/s3.png)
   ![VPC](/images/2/2.1-AWS-Security-Hub/2.1.2-Security-Hub-Dashboard/s3b.png)
4. Scroll down to see graphs for **Most common threat type** and **Software vulnerabilities with exploits**. These visuals help you understand threats and vulnerabilities in your account.
   ![VPC](/images/2/2.1-AWS-Security-Hub/2.1.2-Security-Hub-Dashboard/s4.png)

5. You can also see details about findings from AWS GuardDuty. Click on one to learn more:
   ![VPC](/images/2/2.1-AWS-Security-Hub/2.1.2-Security-Hub-Dashboard/s5.png)
