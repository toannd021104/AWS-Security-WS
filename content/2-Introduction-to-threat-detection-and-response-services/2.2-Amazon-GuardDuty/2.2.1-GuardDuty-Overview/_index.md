## GuardDuty - Overview

Amazon GuardDuty is your AWS security detective! It keeps an eye out for any suspicious or unauthorized activity in your cloud environment. Using smart technology like machine learning and anomaly detection, GuardDuty helps protect your accounts, workloads, and data by analyzing huge amounts of AWS events and logs. You don’t need to install extra software—just let GuardDuty do the heavy lifting so you can focus on keeping your organization secure and innovative.
