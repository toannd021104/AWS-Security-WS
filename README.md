


# AWS Security Workshop

This website provides detailed hands-on labs for AWS security services, helping you understand and apply cloud security solutions on AWS.

## Lab Contents

1. **Security Hub**
	- Aggregate and manage security alerts from multiple AWS services.
	- Step-by-step guide to viewing, filtering, creating insights, and handling findings.

2. **GuardDuty**
	- Automatically detect threats based on behavioral analysis and log data.
	- Practice enabling GuardDuty, reviewing findings, and creating custom threat lists.

3. **Inspector**
	- Scan for vulnerabilities on EC2, containers, and other cloud resources.
	- Instructions for configuring, running scans, viewing results, and remediating findings.

4. **Detective**
	- Analyze and investigate security events, trace root causes of incidents.
	- Practice building graphs and analyzing relationships between events.

5. **Partner Integration & Multi-account Aggregation**
	- Connect partner security solutions to Security Hub.
	- Aggregate findings from multiple AWS accounts and regions.

## Objectives
- Help learners understand, configure, and utilize AWS security services.
- Improve cloud security skills through practical, real-world examples.
- Optimize detection, analysis, and incident response workflows.

## Usage
- Explore detailed labs in the `content/` folder.
- Run the website locally: `hugo server -D` and visit http://localhost:1313
- Every code push will automatically build and deploy the site to GitHub Pages via GitHub Actions.
